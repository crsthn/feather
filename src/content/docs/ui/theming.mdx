## Color Variables

| Variable | Description |
| -------- | ----------- |
| `--bg` | Main background color |
| `--surface` | Surface color for cards and containers |
| `--overlay` | Background for overlays like modals |
| `--primary` | Primary brand color |
| `--primary-muted` | Subtle version of primary color |
| `--secondary` | Secondary color for UI elements |
| `--muted` | Subtle background for disabled or secondary elements |
| `--danger` | Color for dangerous actions |
| `--danger-muted` | Subtle version of danger color |
| `--text` | Primary text color |
| `--text-2` | Secondary text color |
| `--text-3` | Tertiary text color for hints |
| `--input` | Border color for input fields |
| `--border` | Color for borders and dividers |
| `--on-primary` | Text color used on primary background |
| `--on-danger` | Text color used on danger background |
| `--scrollbar` | Color for scrollbar |
