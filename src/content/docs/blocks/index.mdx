## Set up Feather UI

Before using Feather Blocks, you'll need to install and set up Feather UI in your project.

Please visit our [Feather UI Documentation](/docs/ui) for complete installation instructions and getting started guide.

Once you have Feather UI properly installed, you'll be ready to use Feather Blocks in your project.
